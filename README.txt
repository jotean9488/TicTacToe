Tic Tac Toe Generalized - 

Required libraries: pygame

To run, use the command line and run "py game.py" or "python game.py". 
You will be prompted to enter the dimensions of the board. Enter them in the form "mxn", where m is the number of rows and n the number of columns.
The board will then appear in a new window, and you can play the game.
